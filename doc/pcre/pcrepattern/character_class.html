In a character class the only metacharacters are \ ^ - [ ] ,

[ must be ended(with ]), else can't compile

If a range that includes letters is used when caseless matching is set, it matches the letters in either case. For example, [W-c] is equivalent to [][\\^_`wxyzabc], matched caselessly

POSIX CHARACTER CLASSES: [:CLASS_NAME:]

  • space: the same as \s ; VT not included before v8.34
  • blank: space and VT;
  • graph: printing characters, excluding space

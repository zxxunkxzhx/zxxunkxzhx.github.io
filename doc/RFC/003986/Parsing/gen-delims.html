Any delim which is known by this doc is a gen-delim, following are the details:
  • : use to delim scheme and hier-part; or occur in authority(before specifying the port)
  • / used in hier-part
  • ? used to start URI query
  • # used to start URI fragment
  • @ used to delim hier-part->Authority->userinfo and hier-part->Authority->host
  • [ and ] used in hier-part->Authority->Host::IP-literal->"[" ( IPv6address / IPvFuture ) "]"
    such as https://[2001:db8:a0b:12f0::1]:8080/index.html

About this Manual

The architecture describes the operation of an Armv8-A(zxxu:pupular profiles includes Application profile, Microcontroller Profile, R Profile, C Profile) and an Armv9-A Processing element (Arm architecture described in this Architecture Reference Manual defines the behavior of an abstract machine, referred to as a processing element, often abbreviated to PE处理单元). and this Manual includes descriptions of:

  • The two Execution states, AArch64 and AArch32
  • In AArch32 state, the A32 and T32 instruction sets; In AArch64 state, the A64 instruction set.

this Manual is not a tutorial for Arm assembler language, nor does it describe Arm assembler language, except at a basic level. To make effective use of Arm assembler language, read the documentation supplied with the assembler being used.

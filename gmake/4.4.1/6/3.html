Substitution References

 ‘$(var:a=b)’ is equivalent to ‘$(patsubst %a,%b,var)’, which replaces every word in $(var) that ends with a by replacing the suffix a with b


Another type of substitution reference has the same form ‘$(var:a=b)’, except that now a must contain a single ‘%’ character. This case is equivalent to ‘$(patsubst a,b,$(var))’.


Computed Variable Names

Variables may be referenced inside the name of a variable. This is called a computed variable name or a nested variable reference. For example,

x = y
y = z
a := $($(x))


defines a as ‘z’
